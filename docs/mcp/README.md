## 🔌 What is MCP?

**Model Context Protocol (MCP)** is a system for extending AI capabilities by connecting models to additional tools and data sources.

In this project, MCP is used to:
- Integrate with GitHub APIs for issue management
- Connect to project-specific tools and services
- Provide context to AI agents working on tasks
- Enable rich interactions between agents and external systems
